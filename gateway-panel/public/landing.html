Features

Everything you need to stay protected

Enterprise-grade payment infrastructure that high-volume WooCommerce merchants rely on.

Intelligent load balancing

Traffic is distributed across your shield pool automatically. No single shield takes all the volume — your operation stays healthy at scale.

Auto circuit breaker

Unhealthy gateway connections are detected and removed from rotation instantly. Self-healing resets happen automatically — no manual intervention needed.

HMAC-SHA256 authentication

Every API request is signed and timestamped. Replay attack protection with 5-minute window enforcement.

AES-256 key encryption

PayPal and Stripe API credentials are encrypted at rest using AES-256. Your keys are never exposed in plaintext.

Multi-gateway support

PayPal and Stripe out of the box. Airwallex coming in Phase 2. One panel to manage all gateways across all sites.

WooCommerce plugins included

Drop-in plugins for your WooCommerce stores. Self-updating, with health heartbeat and zero config needed. Download straight from your dashboard.

Full transaction audit log

Rate limiting per token

FAQ

Questions? We've got answers.

How does OneShieldX keep my store always processing?
OneShieldX sits as a middleware layer between your WooCommerce store and your payment gateways. Our proprietary routing engine manages your gateway connections intelligently — detecting issues before they impact sales and failing over automatically. The exact mechanism is part of our core IP and is not publicly disclosed.
Does my customer see anything different at checkout?
No. The checkout experience is identical to what your customers already see. No redirects, no extra steps, no third-party branding. OneShieldX operates entirely in the background.
How secure are my PayPal and Stripe API keys?
All API credentials are encrypted at rest using AES-256 (Laravel's Crypt facade). They are never stored in plaintext. Every API call between your money site and the gateway panel is signed with HMAC-SHA256 and includes replay-attack protection via timestamp validation.
What happens if a gateway connection fails?
Our built-in circuit breaker automatically detects the issue and removes that connection from rotation in real time. Traffic shifts to healthy connections instantly. The affected connection auto-recovers after a cool-down — no manual action required.
Do I need technical skills to set it up?
Basic WooCommerce knowledge is enough. Install the plugin on each site, paste your token, and you're done. Full step-by-step setup guide is included in your panel. Enterprise plan includes onboarding support.
Is Airwallex supported?
Airwallex is on the roadmap for Phase 2 and is included in the Pro and Enterprise plans. PayPal and Stripe are fully supported today.
